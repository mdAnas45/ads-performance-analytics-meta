# Meta Ads Performance Dashboard (Power BI)

## Overview

This project presents an interactive **Meta Ads Performance Dashboard** built using **Power BI** to analyze and monitor advertising performance across **Facebook and Instagram** platforms. The dashboard provides a clear view of campaign effectiveness, audience engagement, and conversion behavior to support data‑driven marketing decisions.

---

## Project Purpose

The primary goal of this project is to help marketers and business stakeholders:

* Understand how ad budgets translate into engagement and purchases
* Identify high‑performing ad formats and audiences
* Optimize marketing strategy using performance metrics and trends

---

## Tech Stack

* **Power BI** – Data modeling, DAX measures, interactive dashboards
* **Power Query** – Data cleaning and transformation
* **Excel / CSV** – Source data format
* **Marketing Analytics Metrics** – CTR, Conversion Rate, Engagement Rate

---

## Data Source

* Simulated / Sample Meta Ads data
* Platforms covered: **Facebook & Instagram**
* Dataset includes:

  * Budget & Spend
  * Impressions & Clicks
  * Engagements (Likes, Comments, Shares)
  * Purchases
  * Ad Types (Image, Video, Carousel, Stories)
  * Audience attributes (Age, Gender, Country)
  * Time attributes (Month, Week, Hour)

> Note: This project uses sample data for learning and portfolio demonstration purposes.

---

## Business Problem

Marketing teams often face difficulty identifying which ad formats, audiences, and time periods generate the best return on investment. Raw advertising data alone does not clearly reveal actionable insights.

---

## Dashboard Goals

* Track overall ad performance using key KPIs
* Compare ad effectiveness across different ad types
* Analyze audience behavior by age, gender, and location
* Identify peak engagement periods for better ad scheduling
* Support budget optimization and ROI improvement

---

## Key Dashboard Features

* High‑level KPI cards for budget, impressions, clicks, engagement, and purchases
* Conversion Rate (CR), Click‑Through Rate (CTR), Engagement Rate (ER)
* Performance comparison across ad formats
* Audience demographic analysis
* Geographic distribution of engagement
* Time‑based trend analysis (monthly, weekly, hourly)
* Interactive slicers for platform and targeting insights

---

## Key Visuals & Why They Were Used

### KPI Cards

Provide a quick summary of overall campaign performance at a glance.

### Map Visualization (Comments by Country)

Used to identify geographic regions with high user engagement.

### Bar Charts (Age & Gender)

Helps understand which demographic segments interact most with ads.

### Stacked Column Chart (Ad Type vs Engagement)

Compares engagement distribution across different ad formats.

### Line Chart (Comments by Hour)

Identifies peak engagement hours to improve ad scheduling strategies.

### Performance Table

Allows detailed comparison of ad types using CTR, CR, ER, and Purchase Rate.

---

## Key Insights

* Certain ad formats generate higher engagement and conversion rates
* Specific age groups show stronger interaction patterns
* Engagement varies significantly by time and region
* Balanced budget allocation can improve overall campaign efficiency

---

## Conclusion

This dashboard transforms raw Meta Ads data into meaningful insights, enabling smarter marketing decisions. It demonstrates practical use of Power BI for real‑world digital marketing analytics and performance optimization.
